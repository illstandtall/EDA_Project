For Type Prediction & Classification

For comparing all models of ML. It can be used for prediction.

적색왜성, 갈색왜성, 백색왜석, 주계열성, 초거성, 극대거성의 분류로 나뉩니다.

각 분류별로 40개의 데이터이며, 총 240개의 데이터입니다.

별의 온도, 상대밝기, 상대반지름, 절대등급, 색깔, 방출스펙트럼의 속성이 있습니다.

NASA에서 공개한 데이터입니다.

kaggle 데이터 보기

별의 속성에 대한 평행좌표그래프

1. 온도의 분포는 아주 다양합니다.

2. Type 4, 5를 제외한 상대 밝기는 0에 가깝습니다.

3. Type 5를 제외한 상대 반지름은 0에 가깝습니다.

4. 절대 등급은 작은 값이 더 밝은 별이기 때문에

Type의 순서와 반대임을 알 수 있습니다.

상대 밝기(L)와 상대 반지름(R) 분포

1. 대부분의 분류의 데이터가 상대 밝기는 많이 차이나지만,

상대 반지름은 거의 비슷한 것을 알 수 있습니다.(수평으로 분포)

2. 분류별로 데이터 값의 분포가 극단적으로 차이가 납니다.

(0,0) 좌표 근처를 확대해보면

Type 0, 1, 2의 별들이 모여 있음을 알 수 있습니다.

절대 등급(A_M)과 온도(Temperature)의 분포

1. Type 0에서 Type 5로 갈 수록 (왼쪽에서 오른쪽)

절대등급이 낮아지는 것을 알 수 있습니다.

따라서, 절대등급을 통해 어떤 분류인지 대략적으로 알 수 있습니다.

2. Type 3, 4, 5는 분류 내에서 온도의 분포는 다양하지만,

절대 등급은 비슷한 분포를 보입니다.

따라서, 온도를 통해 별의 분류를 알기 어렵습니다.

절대 등급(A_M)과 온도(Temperature), 상대 밝기(L)의 분포
